### 1.2 TopBar组件

| 区域 | 内容 | 数据源 | 更新频率 |
|------|------|--------|---------|
| 左侧 | BIOCore Logo + 版本号 | 静态 | — |
| 中央 | 反应器状态摘要：`F01 ●Running Phase 4/9 Step 2/4 [就地灭菌·升温至121°C]` | WebSocket: state_update | 每秒 |
| 中央(次) | 运行时长计时器：`已运行 02:34:17` | 本地计时器 | 每秒 |
| 中央(右) | **通讯状态指示**: ●绿(正常) / ●红闪(断线) / ●黄(重连中) | WebSocket: heartbeat | 每秒 |

| 右侧 | 用户头像+角色 | SQLite: users | 登录时 |

| 状态 | 颜色 | 动画 |

| Held | 橙色 🟠 | 快闪 |
| Paused | 蓝色 🔵 | 无 |
| Stopped | 红色 🔴 | 无 |

### 2.2 控制面板——状态机控制区

**状态指示区：**

**操作按钮组布局与使能规则：**

```
[  ▶ 开始  ]  [  ⏸ 暂停  ]  [  ⏹ Hold  ]     ← 第一行
[  ▶ 继续  ]  [  ↺ 恢复  ]  [  ■ 停止  ]     ← 第二行
[  ⟲ 复位  ]                                   ← 第三行
[       ⚠ 急停       ]                         ← 第四行（红色大按钮，独占一行）
```

| 按钮 | 命令 | 颜色 | 使能条件 | 确认弹窗 |
|------|------|------|---------|---------|
| ▶ 开始 | cmd_start | 绿色 | state=idle | 是（选择配方+确认参数） |
| ⏸ 暂停 | cmd_pause | 蓝色 | state=running | 否 |
| ⏹ Hold | cmd_hold | 橙色 | state=running | 否 |
| ▶ 继续 | cmd_unpause | 蓝色 | state=paused | 否 |
| ↺ 恢复 | cmd_restart | 绿色 | state=held | 是（确认故障已排除） |
| ■ 停止 | cmd_stop | 红色 | state=held OR paused | 是（确认终止批次） |
| ⟲ 复位 | cmd_reset | 灰色 | state=stopped OR complete | 是（确认清除） |
| ⚠ 急停 | safety_estop | 红底白字 | state≠stopped AND state≠complete | 是（二次确认） |

### 2.3 Phase/Step状态区

```
┌─────────────────────────────────────┐
│  Phase 进度                          │
│                                     │
│  ●完成 ●完成 ●完成 ◉当前 ○ ○ ○ ○ ○  │
│  准备  加水  加料  SIP  控温接种发酵发酵出料│
│  (1)  (2)  (3)  (4) (5)(6)(7)(8)(9) │
│                                     │
│  当前Phase: 就地灭菌 (sip)           │
│  ┌─────────────────────────────┐   │
│  │ Step 1 预排空气       ✓ 完成 │   │
│  │ Step 2 升温至121°C    ▶ 进行中│   │
│  │   AI-0: 108.3°C / 121°C     │   │
│  │   AI-1: 105.7°C / 121°C     │   │
│  │   ██████████░░░░  89%        │   │
│  │ Step 3 保温F₀积分     ○ 待执行│   │
│  │ Step 4 冷却           ○ 待执行│   │
│  └─────────────────────────────┘   │
└─────────────────────────────────────┘
```

**Step进度条计算方式：**

| 完成条件类型 | 进度计算 |
|------------|---------|
| `>=` | (PV - 起始值) / (目标值 - 起始值) × 100% |
| `<=` | (起始值 - PV) / (起始值 - 目标值) × 100% |
| `in_band` | 偏差越小越接近100% |
| `duration` | elapsed / target_seconds × 100% |
| `accumulated` | 累积值 / 目标值 × 100% |
| `delta` | (当前delta / 目标delta) × 100% |
| `and` | 所有子条件进度的最小值 |
| `or` | 所有子条件进度的最大值 |

### 2.4 实时参数卡片

每张卡片统一结构：

```
┌───────────────────┐
│ 温度          AI-0 │  ← 参数名 + 通道号
│                    │
│   37.2°C          │  ← PV值（大字）
│   SV: 37.0°C      │  ← 设定值
│   CV: 45%  加热    │  ← 控制输出 + 当前模式
│   ████████░░ 0.2°C │  ← 偏差条 + 偏差值
└───────────────────┘
```

**偏差条颜色：**

| 偏差范围 | 颜色 |
|---------|------|
| \|PV-SV\| ≤ 死区 | 绿色 |
| 死区 < \|PV-SV\| ≤ 死区×3 | 黄色 |
| \|PV-SV\| > 死区×3 | 红色 |

### 2.5 实时趋势图区

Plotly.js渲染，所有图表共享时间轴（最近60分钟滑动窗口），支持鼠标框选缩放。

| 图表 | Y轴 | 曲线 |
|------|-----|------|
| 温度 | °C (0~150) | 罐温PV(蓝实线), 罐温SV(蓝虚线), 夹套温度(灰实线) |
| pH | pH (3~10) | pH PV(紫实线), pH SV(紫虚线) |
| DO | % (0~100) | DO PV(绿实线), DO SV(绿虚线), AI估算OD(灰虚线+置信带, v2.0) |
| 搅拌+通气 | 双Y轴: rpm / NL/min | 搅拌(橙实线), 通气(青实线) |
| 补料+碱耗 | 双Y轴: mL/h / mL累积 | 补料速率(红实线), 累积补料(红虚线), 碱消耗速率(蓝实线), 累积碱耗(蓝虚线) |

**AI软测量值(v2.0)显示规范：**

| 要素 | 硬件传感器 | AI软测量 |
|------|----------|---------|
| 线型 | 实线 2px | 虚线 1.5px |
| 颜色 | 主题色 | 灰色 #999 |
| 置信区间 | 无 | 半透明色带 (alpha=0.15) |
| 标签 | `DO: 22.3%` | `AI估算OD: 12.3 ±1.5 (R²=0.91)` |
| 图例标注 | — | 前缀"AI·" |

```

│ 💡 AI建议 (feed-advisor)            │
│                                     │
│ 建议将补料速率从 12.3 mL/h          │

│                                     │
```

### 3.2 多批次叠加

叠加后：当前批次用实线+主题色，历史批次用半透明(alpha=0.3)+灰色调，时间轴对齐到发酵经过时间（非绝对时间）。

**Phase类型选择器分组：**

| 分组 | Phase类型 |
|------|----------|
| 准备操作 | 准备、加水、人工加料 |

| 收尾清洗 | 出料、清洗CIP |

**参数表单：** 根据选中Phase的type动态渲染。发酵Phase包含嵌套的controls和DO策略子表单。

**固定步骤序列展示：** 在参数区上方只读显示当前Phase类型的固定Step列表，工艺工程师可了解执行流程但不允许编辑。

## 八、WebSocket数据流

### 8.1 频道定义

| 频道 | 推送频率 | 数据内容 | 消费组件 |
|------|---------|---------|---------|
| `heartbeat` | 每秒 | PC计数器 + PLC计数器 + 通讯状态(alive/loss/reconnecting) + 丢包率 | TopBar通讯指示灯 |
| `pv_realtime` | 每秒 | 全部AI通道PV + VFD转速 + 阀门CV | 参数卡片, 趋势图 |
| `state_update` | 事件驱动 | 状态机状态 + Phase/Step + 按钮使能 | 控制面板, TopBar |
| `step_progress` | 每秒 | 当前Step完成条件进度(%) | Phase/Step状态区 |
| `calculated` | 每分钟 | OUR, kLa, μ, 累积值 | 底部测算横条 |
| `alarm` | 事件驱动 | 新报警/确认 | 报警横幅, TopBar badge |
| `ai_suggestion` | 事件驱动 | 新AI建议/过期 | AI建议卡片 |
| `soft_sensor` | 每分钟(v2.0) | OD/葡萄糖估算+置信区间 | 趋势图AI曲线 |
| `cusum` | 每分钟 | 各参数CUSUM偏差和报警状态 | AI助手上下文面板 |
