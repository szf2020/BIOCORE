# 08 BIOCore 实验设计策略

> 参考：《试验设计与数据处理》方法体系  
> 适用：BIOCore 5L 发酵罐控制平台  
> 版本：v1.0 | 2026-04-10

---

## 1 概述

本文档基于《试验设计与数据处理》的方法体系（正交设计、方差分析、回归建模、均匀设计），为 BIOCore 平台制定三阶段实验策略。

**目标：**
1. 发酵工艺优化 — 温度/pH/DO/转速/补料的最优组合
2. PLC 控制参数调优 — 8 路 PID 的 Kp/Ki/Kd 整定
3. 平台整体验证 — 可靠性、精度、重现性确认

**总计 ~47 次实验，约 14 周。**

---

## 2 因素与指标定义

### 2.1 可控因素

| 因素 | 代号 | PLC 标签 | 实验范围 | 单位 |
|------|------|----------|----------|------|
| 温度 | A | TEMP_SV (VD10) | 25 – 42 | °C |
| pH | B | PH_SV (VD20) | 5.5 – 8.0 | pH |
| 溶氧设定 | C | DO_SV (VD16) | 10 – 60 | % |
| 搅拌转速 | D | RPM_SV (VW14) | 200 – 800 | rpm |
| 通气量 | E | AIRFLOW_SV (VD28) | 2 – 20 | NL/min |
| 补料策略 | F | P02 pump | 恒速/指数 | — |

### 2.2 响应指标

| 指标 | 代号 | 数据来源 | 优化方向 |
|------|------|----------|----------|
| 产物浓度 | Y1 | offline_samples.product_titer | 最大化 |
| 体积生产率 | Y2 | Y1 / 发酵时长 | 最大化 |
| 最大菌密度 | Y3 | offline_samples.OD600 max | 最大化 |
| 比生长速率 | Y4 | calculated_params.mu max | 参考 |
| 传质系数 | Y5 | calculated_params.kLa | 参考 |
| 乙酸积累 | Y6 | offline_samples.acetate_g_L | 最小化 |

### 2.3 多指标综合评分

```
W = 0.35×Y1_norm + 0.25×Y2_norm + 0.20×Y3_norm − 0.20×Y6_norm
```

归一化方法：  
- 最大化指标：`norm = (Y − Y_min) / (Y_max − Y_min)`  
- 最小化指标：`norm = (Y_max − Y) / (Y_max − Y_min)`

## 3 Phase I：筛选与单因素优化（第 1-4 周）

> 书中方法：第 2 章黄金分割法 + 第 3 章单因素方差分析

### 3.1 目的

确定哪些因素对关键指标影响显著，缩小各因素的实验范围。

### 3.2 基线条件

| 参数 | 值 |
|------|-----|
| 温度 | 37°C |
| pH | 7.0 |
| DO | 30% |
| 转速 | 400 rpm |
| 通气量 | 10 NL/min |
| 补料 | 恒速 15 mL/h, 400g/L 葡萄糖 |
| 接种密度 | OD600 = 0.3 |

### 3.3 黄金分割单因素搜索（22 次实验）

对 A-E 每个因素独立搜索，其余固定在基线。

**黄金分割比：** 0.618

**API 调用：** `POST /api/v1/doe/golden-section/next`

```json
{
  "factorName": "temperature",
  "min": 25, "max": 42,
  "goal": "maximize",
  "evaluatedPoints": []
}
```

**因素 A — 温度（5 次，每次 24h）：**

| 编号 | 温度 (°C) | 计算 |
|------|-----------|------|
| A1 | 25.0 | 下界 |
| A2 | 42.0 | 上界 |
| A3 | 35.5 | 42 − 0.618×17 |
| A4 | 31.5 | 25 + 0.618×17 |
| A5 | 缩小区间 | 根据 A3/A4 结果 |

其他因素同理。D×E 交互用 2² 析因（4 次）。

### 3.4 重复实验 + ANOVA（3 次实验）

在搜索确定的最佳条件下重复 3 次：

- 单因素 ANOVA：F = MS_between / MS_within，α=0.05
- 排序因素显著性
- 建立 CUSUM 基线

### 3.5 Phase I 产出

1. 因素显著性排名
2. 各因素缩小后的实验范围
3. CUSUM 异常检测基线
4. 初始批次包络线

## 4 Phase II：多因素正交优化（第 5-9 周）

> 书中方法：第 4-6 章正交设计 + 极差/方差分析，第 7 章部分追加法，第 9 章均匀设计

### 4.1 正交试验 L9(3⁴)（9 次实验）

取显著性最高的 4 个因素，每因素 3 水平。

**API 调用：** `POST /api/v1/doe/orthogonal/generate`

```json
{
  "factors": [
    { "name": "温度", "levels": [34, 37, 40], "unit": "°C" },
    { "name": "pH", "levels": [6.5, 7.0, 7.5], "unit": "pH" },
    { "name": "DO", "levels": [20, 30, 40], "unit": "%" },
    { "name": "转速", "levels": [300, 500, 700], "unit": "rpm" }
  ]
}
```

**L9(3⁴) 方案：**

| 编号 | 温度 | pH | DO% | 转速 |
|------|------|-----|-----|------|
| OA1 | 34 | 6.5 | 20 | 300 |
| OA2 | 34 | 7.0 | 30 | 500 |
| OA3 | 34 | 7.5 | 40 | 700 |
| OA4 | 37 | 6.5 | 30 | 700 |
| OA5 | 37 | 7.0 | 40 | 300 |
| OA6 | 37 | 7.5 | 20 | 500 |
| OA7 | 40 | 6.5 | 40 | 500 |
| OA8 | 40 | 7.0 | 20 | 700 |
| OA9 | 40 | 7.5 | 30 | 300 |

每次实验对应一个 BIOCore 配方。

### 4.2 极差分析

**API 调用：** `POST /api/v1/doe/orthogonal/analyze`

分析内容：
- K_j = 因素在水平 j 下的 Y 值之和
- k_j = K_j / 3
- R = max(k_j) − min(k_j)
- 按 R 排序因素重要性
- 最优水平组合

### 4.3 方差分析

- SS_total = SS_A + SS_B + SS_C + SS_D + SS_error
- 饱和 L9 → 合并法（最小 SS 因素并入误差）
- F 检验，α=0.05 和 α=0.01

### 4.4 补料策略子优化（部分追加法，3 次实验）

取正交最优组合，单独变化补料：

| 编号 | 补料策略 |
|------|----------|
| FS1 | 恒速 10 mL/h |
| FS2 | 恒速 25 mL/h |
| FS3 | 指数补料 μ=0.15 |

### 4.5 PID 参数均匀设计（3-4 次实验）

重点调优：温度 (PID01)、DO 级联 (PID04)、pH (PID06)。

**API 调用：** `POST /api/v1/doe/uniform/generate`

```json
{
  "factors": [
    { "name": "Kp", "min": 4.0, "max": 14.0 },
    { "name": "Ki", "min": 0.1, "max": 1.5 },
    { "name": "Kd", "min": 0.5, "max": 5.0 }
  ],
  "tableName": "U7(7³)"
}
```

**U7(7³) 方案示例（Z-N 估计 Kp₀=8, Ki₀=0.5, Kd₀=2）：**

| 编号 | Kp | Ki | Kd |
|------|-----|------|------|
| PID-T1 | 4.0 | 0.3 | 3.0 |
| PID-T2 | 5.0 | 0.7 | 0.5 |
| PID-T3 | 6.0 | 1.5 | 2.0 |
| PID-T4 | 8.0 | 0.1 | 5.0 |
| PID-T5 | 10.0 | 0.5 | 1.5 |
| PID-T6 | 12.0 | 1.0 | 1.0 |
| PID-T7 | 14.0 | 0.2 | 4.0 |

PID 调优指标：调节时间、超调量、IAE、控制能耗。

## 5 Phase III：验证与建模（第 10-14 周）

> 书中方法：第 8 章回归分析

### 5.1 回归建模

**API 调用：** `POST /api/v1/doe/regression/fit`

利用全部实验数据拟合二阶响应曲面：

```
Y = b₀ + Σbᵢxᵢ + Σbᵢᵢxᵢ² + Σbᵢⱼxᵢxⱼ + ε
```

4 因素 → 15 个系数，目标 R² > 0.85。

### 5.2 确认实验（5 次重复）

| 编号 | 目的 |
|------|------|
| V1-V3 | 标准重复，95% 置信区间 |
| V4 | 长时间 48h 可持续性验证 |
| V5 | 贝叶斯优化微调点 |

**验收标准：**
- |观测值 − 预测值| < 2×SE
- 产物浓度 CV < 5%
- 温度控制 CV < 3%

### 5.3 PID 性能验证

| PID 回路 | 测试内容 | 验收标准 |
|----------|----------|----------|
| 温度 PID01 | 37→38→37°C 阶跃 | 调节时间 <300s，超调 <0.5°C |
| DO PID04 | 30→40→30% 阶跃 | 调节时间 <120s，超调 <5% |
| pH PID06 | 7.0→6.8→7.0 阶跃 | 调节时间 <180s，超调 <0.1 pH |

### 5.4 平台可靠性指标

| 类别 | 指标 | 目标 |
|------|------|------|
| 控制精度 | MAE = mean(\|PV−SV\|) | — |
| 带内率 | \|PV−SV\| < deadband 时间占比 | >95% |
| 数据完整性 | InfluxDB 写入成功率 | >99.9% |
| 重现性 | 批次间 CV | <5% |
| 安全响应 | 通讯断线→Hold 延迟 | <3s |
